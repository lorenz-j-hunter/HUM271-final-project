### Goal
The app aims to get data on women's perception of masculinity within society.

With this site, a user will be able to (1) select a website to scrape and (2) create a CSV which contains
columns. One column (nodes) will be the authors of posts. The second column (gender) will be the gender of
said author. The third column (edges) will be the feed they are in, or the recipient of the message. The third
column (edge width) will be the overall expressed disdain of masculinity in particular within the text.

### Methodology
A user should be able to scrape post data from a website they choose, then export it as a CSV.
